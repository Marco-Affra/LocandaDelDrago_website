# 🐉 La Locanda del Drago - Digital Menu

Benvenuti nel repository ufficiale della **Locanda del Drago**. Questo è un sito web moderno, veloce e responsive sviluppato in **React** per permettere ai clienti della locanda di consultare il menù digitale tramite QR Code o link diretto.

---

## 🚀 Demo Live
🔗 [Visualizza il sito live su Netlify](https://locandadragoristorante.it/) 

---

## ✨ Caratteristiche
- **React-Powered:** Interfaccia fluida e caricamento istantaneo dei piatti.
- **Mobile First:** Ottimizzato per la visualizzazione su smartphone (ideale per i clienti al tavolo).
- **Menu Dinamico:** Suddivisione chiara tra Antipasti, Primi, Secondi e Bevande.
- **Accessibilità:** Design ad alto contrasto per una lettura agevole in ogni condizione di luce.
- **Deploy Automatico:** Grazie all'integrazione con Netlify, ogni modifica su questo repository si riflette istantaneamente online.

---

## 🛠️ Tecnologie Utilizzate
- **Frontend:** [React.js](https://reactjs.org/)
- **Styling:** CSS3 / Styled Components (o la libreria che stai usando)
- **Hosting & CI/CD:** [Netlify](https://www.netlify.com/)
- **Domain Management:** [Hosting Aruba](https://hosting.aruba.it/)
